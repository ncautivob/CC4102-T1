- 📄 $\texttt{search.cpp}$ - Método de búsqueda generalizado para ambos métodos de construcción.
- 📄 $\texttt{informe.pdf}$ - Informe de la tarea en formato PDF.
- 📄 $\texttt{tarea-1.cpp}$ - Archivo principal de la tarea.
